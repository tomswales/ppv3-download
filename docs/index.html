Payment Widget Version Manager

About this tool

This tool is for merchants using the Payoneer Checkout Payment Widget in their online shop who want to know what versions of the payment widget are available and to download the latest widget versions.

Current payment widget versions

The table below shows which versions are currently the latest released on our Test (Sandbox) and Live environments. Any version currently on Test (Sandbox) but not Live should only be tested against the Test (Sandbox) environment
